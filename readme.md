# Happy gitting everyone!

* fork this repo (use the GH fork button)
* clone the result to your computer  
  `git clone <url>`
* make some edits  
  you can use any text editor
* commit your edits  
  `git commit -am "<describe what did your do>"`
* push edits back to gh!  
  `git push`
